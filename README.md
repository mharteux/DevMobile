# DevMobile
 Desenvolvimento de dispositivos móveis

Macs commands

Create project
npx create-expo-app@latest MeuPrimeiroApp --template blank

Project working on web
npx expo install react-native-web react-dom @expo/metro-runtime

Project running
npm run web
npm run ios
